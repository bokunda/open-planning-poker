About

What is Open Planning Poker?

Open Planning Poker is a free and open-source solution that implements one of the ways how to plan and estimate effort. The idea is simple, you have to import or write all the user stories/features/tasks and during the 'poker' sessions you have to discuss about the scope and effort of each item as a team. The next step is voting where you have to assign a complexity value for the discussed story. When the voting is finished, an average value is calculated and then you can argue if the average value is acceptable or to vote again.

Features

Open Planning Poker provides you with an easy option to:

  1. Create a poker room.
  2. Add stories/features/tasks
  3. Voting mechanism
  4. Basic voting results

How to play?

When the page opens, you will be assigned a random username that can be changed.

You can create a game, or join the existing one using a URL.

Voting is possible only when a ticket is created.

Privacy

Open Planning Poker is an open source project so you can check the source code. Users are deleted 24h after creation. Games, ticktes and voting related data is stored longer, because it can be useful for statistics, and new features.
