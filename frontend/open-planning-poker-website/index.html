Open Planning Poker is a collaborative agile estimation tool designed to streamline your sprint planning sessions. It's an engaging and effective way for development teams to estimate the effort required for each user story or task in a project.

The team gathers to discuss and estimate the complexity of user stories. Each team member privately selects an estimation value using a deck of cards. The selected values are revealed simultaneously, fostering discussion and convergence. Repeat until a consensus is reached for each user story.

Open Planning Poker is a free, open source, project. You can use the application on our server or you can host it by yourself.

For all change requests, bugs, ideas.. feel free to create an issue on GitHub, or create a pull request with a code change.
